About roytang.net

  • roytang.net is a personal site, an E/N site, and kind of a commonplace book
  • I post about a random assortment of topics that interest me including software development, Magic the Gathering, pop culture, gaming, and tech life. This site is perpetually under renovation.
  • I’ve had a site on the internet since around 2001, in some form or another.
  • I’ve had the domain roytang.net since 2006.
  • You can view a changelog of this site.
  • The site is perpetually under renovation, as we all are.
  • This site is statically generated using Hugo with a custom theme that I internally call “Exhale” for some reason.
  • The source for this website is available on Github.
  • Dynamic content on this website is supported by a ragtag collection of Python scripts and third-party services.
    • Comments are handled via a custom comments backend written in Django.
    • Search is handled via a custom search backend using Django and Whoosh.
    • Webmentions are handled via webmention.io + brid.gy (incoming) and webmention.app (outgoing).
    • content I post on Twitter, Instagram and other services are fed back into this site via custom Python scripts.
  • This site is hosted on Webfaction (affiliate link).
  • An off-site mirror/backup is generated every so often to https://roytang.github.io/.
